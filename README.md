# マーケ実践コックピット

えいちゃん専用のマーケ学習PWAです。

## v2の範囲

- STEP1 根っこ: マーケの本質、情熱、煽り売り回避
- STEP2 コンセプト・商品設計: 顧客の生の声DB、痛み止め商品、悩みから理想まで
- STEP3 集客: 媒体役割、一次情報、入口投稿、PASONA
- STEP4 リスト化と教育: 売上方程式、無料/購入者リスト、公式LINE、オプチャ、メルマガ、Substack
- STEP5 オファー: 売ってから作る、自然なオファー、順番の確認

## 使い方

`index.html` をブラウザで開きます。

スマホやタブレットでは、ブラウザの共有メニューからホーム画面へ追加できます。

## GitHub Pages 公開手順

1. GitHubで `marketing-cockpit` リポジトリを作る
2. このフォルダの中身だけをアップロードする
3. `Settings` → `Pages` を開く
4. `Deploy from a branch` を選ぶ
5. `main` / `root` を選んで保存する
6. 表示されたURLをスマホで開き、ホーム画面へ追加する
